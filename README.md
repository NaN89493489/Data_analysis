## Описание проекта

Данный проект выполняет анализ терминов на основе их векторных представлений.  
Основная цель — обработать данные, снизить размерность векторного пространства, выполнить кластеризацию терминов и визуализировать результаты.

Проект включает следующие этапы обработки данных:

1. Чтение JSON-файла с векторными представлениями терминов.
2. Фильтрация терминов по списку запрещённых слов.
3. Снижение размерности векторного пространства до 2D с использованием алгоритма UMAP.
4. Расчёт статистических метрик и рангов.
5. Кластеризация терминов методом K-means.
6. Визуализация полученных кластеров и распределения метрик.

## Используемые алгоритмы и технологии

В проекте используются следующие библиотеки:

- pandas
- numpy
- json
- sklearn.cluster
- umap
- seaborn
- matplotlib
- os

### UMAP
Алгоритм UMAP используется для снижения размерности векторного пространства и получения двумерных координат для каждого термина.

### K-means
Для кластеризации был выбран алгоритм K-means. 
Алгоритм позволяет получить устойчивые группы объектов, хорошо подходит для работы с данными после алгоритма UMAP и при небольшом количестве данных результаты кластеризации удобно визуализировать и анализировать.
Количество кластеров было выбрано "методом локтя". 

## Структура проекта

project/

- pipeline.py - класс для обработки данных
- main.py - точка входа программы
- requirements.txt - зависимости проекта
- data.json - исходные данные
- bad_words.csv - список запрещённых слов
- export/ - папка с результатами выполнения

## Запуск программы

Для запуска проекта необходимо выполнить:

python main.py

## Результаты работы

После выполнения программы автоматически создаётся папка `export`, в которой содержатся следующие файлы:

- **json_data.csv** — исходные данные из JSON
- **filter_data.csv** — данные после фильтрации
- **statistic_data.csv** — данные со статистикой и рангами
- **cluster_data.csv** — данные с присвоенными кластерами
- **cluster_plt.png** — график кластеризации терминов
- **metric_plt.png** — график распределения метрик по кластерам
